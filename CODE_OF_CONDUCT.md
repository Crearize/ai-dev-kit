# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

Instances of unacceptable behavior may be reported by opening a [GitHub Issue](https://github.com/Crearize/ai-dev-helm/issues) with the label `conduct`.

Community leaders will review and respond to reports in a fair and timely manner.
