# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in ai-dev-helm, please report it through [GitHub Security Advisories](https://github.com/Crearize/ai-dev-helm/security/advisories/new).

**Please do NOT report security vulnerabilities through public GitHub issues.**

## Response Timeline

- We will acknowledge receipt of your report within **14 business days**
- We will provide an assessment and planned fix timeline after investigation

## Disclaimer

This project is maintained on a best-effort basis. We do not guarantee emergency response or immediate fixes. Security issues will be addressed as quickly as reasonably possible, but no specific SLA is provided.

## Scope

This security policy applies to the latest released version of `@crearize/ai-dev-helm` on npm.
